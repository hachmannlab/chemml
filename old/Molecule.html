Molecule¶

The central class to construct a molecule from different chemical input formats. This module is built on top of RDKit and OpenBabel python API. We join the forces and strength of these two cheminformatic libraris for a consistent user experience.

Almost all the molecular descriptors and molecule-based ML models require the chemical informatin as a Molecule object. Several methods are available in this module to facilitate the manipulation of chemical data.

Import module¶

[1]:

from chemml.chem import Molecule

We play with caffeine molecule in this tutorial. Here are three representation formats of this molecule:

[2]:

caffeine_smiles = 'CN1C=NC2=C1C(=O)N(C(=O)N2C)C'
caffeine_smarts = '[#6]-[#7]1:[#6]:[#7]:[#6]2:[#6]:1:[#6](=[#8]):[#7](:[#6](=[#8]):[#7]:2-[#6])-[#6]'
caffeine_inchi = 'InChI=1S/C8H10N4O2/c1-10-4-9-6-5(10)7(13)12(3)8(14)11(6)2/h4H,1-3H3'

Construct a molecule¶

You can construct a molecule by instantiating the Molecule class with two required parameters: - input: representaion string or file name - input_type: as of now only smiles, smarts, inchi, and xyz formats are available

[3]:

mol = Molecule(caffeine_smiles, input_type='smiles')

You can always see a representation of the available attributes and chemical formats. The creator shows you how this molecule was originally constructed. The rdkit and pybel molecule objects are also accessible if they have been used to create the molecule.

Visualize¶

The visualize method show or save a molecule image.

[7]:

mol.visualize()

[7]:

You can add or remove hydrogens by:

[8]:

mol.hydrogens('add')
mol.visualize()

[8]:

For example, you need to add hydrogens if you want to optimize the geometry of this molecule to get the spatial informatin.

Convert chemical formats¶

You can use to_smiles, to_smarts, to_inchi, or to_xyz methods with available parameters to convert the formats, accordingly.

[9]:

mol.hydrogens('remove') # just for the sake of visualization: to fit the text inside the box
mol.to_smiles(kekuleSmiles=True) # although I already had it: just to make sure I have kekulized SMILES
mol.to_smarts(isomericSmiles=False)
mol.to_inchi()

Optimize XYZ geometry¶

You can optimize a molecule’s structure using molecular simulation and force fields via to_xyz method. If the 3D info is not available you should specify a force field as an optimizer.

[10]:

# mol.to_xyz(optimizer=None)
# at this point of the script, the above code raises the following error message:
# ValueError: The conformation has not been built yet (maybe due to the 2D representation of the creator).
# You should set the optimizer value if you wish to embed and optimize the 3D geometry.

[11]:

mol.hydrogens('add')
mol.to_xyz(optimizer='MMFF', mmffVariant='MMFF94s', maxIters=300) # 'UFF'

Read XYZ file¶

We utilize Pybel file parsers to build a molecule from chemical input files and convert them to other formats.

[13]:

mol = Molecule('images/1_opt.xyz', 'xyz')
mol

[13]:

<chemml.chem.Molecule(
        rdkit_molecule : None,
        pybel_molecule : <pybel.Molecule object at 0x117a73f60>,
        creator        : ('XYZ', 'images/1_opt.xyz'),
        smiles         : None,
        smarts         : None,
        inchi          : None,
        xyz            : <XYZ(geometry: (28, 3), atomic_numbers: (28, 1), atomic_symbols: (28, 1))>)>

[14]:

mol.visualize()

[14]:

You can create 2D chemical representations using previous methods. Note that the rdkit molecule will be created for this conversion becuase it is our priority to manipulate 2D chemical info.

[15]:

mol.to_smiles(canonical=True)
mol.to_smarts()
mol.to_inchi()
mol
